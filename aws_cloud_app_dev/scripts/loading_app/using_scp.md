# loading_pre_built_app

## from local terminal
```bash
scp -i 'path to key' 'path to file' 'ubuntu@public_id:path in vm'
```
## unzip the files
- sudo apt install unzip: install unzipping tools
- unzip file

## download dependencies
- npm install
- npm start: runs the app
- add :3000 to end of public DNS to access app
